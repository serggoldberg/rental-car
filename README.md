# RentalCar

A web application for browsing and booking rental cars.

## Features

- Browse available cars with filters by brand, price, and mileage
- Infinite scroll pagination (Load More)
- Detailed car page with specifications and rental conditions
- Booking form with validation

## Tech Stack

Next.js 15, TypeScript, TanStack Query, CSS Modules

## Getting Started

```bash
npm install
npm run dev

Open http://localhost:3000
```
